# 🚀 Getting started with Strapi

基于strapi快速构建的测试项目

### `develop`

首先确认电脑上所安装的node.js版本大于18小于22

```
npm install
# or
pnpm install
```

### `start`

可通过一下命令进行启动

```
npm run dev
# or
pnpm run dev
```

打开http://localhost:1337进入登录页面

登陆后可在Content-Type Builder 处创建自己的内容

可通过url方式对api进行访问http://localhost:1337/api/hero-sections/
